A modern Android Bus Booking and Tracking Application for Pakistan.

**About the App**
Bus Booking App is a smart Android application that allows passengers to search routes, view available buses, select seats, and manage their bookings easily.

The application is designed to simplify bus travel by providing route information, seat reservation, booking management, and travel details in a clean and user-friendly interface.

Users can search routes between major Pakistani cities, view bus schedules, select seats visually, and manage upcoming or completed bookings.

**Features**

**Home Screen**
- Select departure city
- Select destination city
- Select travel date
- Search available buses
- View popular routes

**Routes Screen**
- Search routes
- View distance information
- View estimated travel time
- View route fare in PKR
- View available buses

**Seat Booking System**
- Interactive seat layout
- Available seats
- Selected seats
- Booked seats
- Multiple seat selection
- Real-time fare calculation

**My Bookings**
- Upcoming bookings
- Completed bookings
- Booking details
- Seat information
- Travel route information
- Ticket fare details
- Cancel booking option
- View ticket option

**Additional Features**
- Firebase Integration
- Modern UI Design
- RecyclerView Implementation
- Material Design Components
- Responsive Layout
- Clean User Experience
